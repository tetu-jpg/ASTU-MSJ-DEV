My color Palete Picker
Color palette picker is done by react which help us to change the color of the container when ever abutton is clicked.
How it works?

when one of the colored button is clicked, it changes the color to its color and then displays " the selected color is {color}" and have a reset button to reset to it's placeholder.

Tech used:
React-vite: very popular in developing UI.
Tailwindcss: a css extension
